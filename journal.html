Weekly Journal Entries

Week 1 - Introduction to Mobile Development

Tools and Setup

In week 1 we established the development workflow and learned basic version control with Git and GitHub. We set up our project using VS Code and explored simple deployment options. We also experimented with creating a progressive web app (PWA) shell and discussed how backend services could support a mobile app.

First learning

Using PythonAnywhere we experimented with a temperature-converter PWA to understand offline caching and installability. The goal was to get a small, working PWA and learn how service workers and manifests help offline behavior.

Week 2 - Frontend Fundamentals

learning journal skeleton

This week we built the skeleton of the Learning Journal PWA using only HTML and CSS, following a mobile-first approach. Focus areas included semantic HTML structure, responsive layout with Flexbox/Grid, and a central stylesheet for consistent styling across pages.

What I learned

The most useful concepts were mobile-first design and using CSS variables for consistent theming. I practiced structuring pages (index, journal, about, projects) and making a reusable navigation layout.

Week 3 - Next steps

Next week we will begin adding JavaScript for interactivity: DOM manipulation, event handling, and progressively enhancing the PWA with dynamic behavior.
